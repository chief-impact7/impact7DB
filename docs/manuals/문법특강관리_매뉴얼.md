# 문법 특강 관리 매뉴얼

> 컴퓨터에 익숙하지 않은 선생님도 따라할 수 있도록 모든 단계를 세세하게 안내합니다.
> 문법 특강은 **비내신 기간**에 진행되는 문법 수업을 일괄 등록하는 기능입니다.

---

## 시작하기 전에 준비할 것

1. **인터넷 연결** 확인 (와이파이 또는 유선)
2. **크롬 브라우저** 사용 권장
3. **승인된 Google 계정**으로 로그인 가능한지 확인
4. **CSV 파일 준비** — 문법 특강에 참여할 학생 명단이 담긴 CSV 파일 (엑셀에서 저장 가능)

---

## CSV 파일이란?

CSV는 엑셀 데이터를 저장하는 간단한 파일 형식입니다.

### 엑셀에서 CSV 만드는 법
1. 엑셀에서 학생 명단을 작성합니다.
2. **파일** → **다른 이름으로 저장**을 클릭합니다.
3. 파일 형식을 **CSV (쉼표로 분리) (*.csv)** 로 선택합니다.
4. **저장** 버튼을 클릭합니다.
5. "일부 기능이 호환되지 않습니다" 같은 경고가 나오면 **예** 를 클릭합니다.

### CSV 파일 필수 양식

엑셀 첫 번째 줄(헤더)에 아래 열 이름을 적어야 합니다:

| 열 이름 | 필수 여부 | 설명 | 예시 |
|---------|----------|------|------|
| **이름** | 필수 | 학생 이름 | 김민준 |
| **연락처** | 필수 | 학부모 전화번호 | 01012345678 |
| **재원/비원** | 선택 | 기존 학생 여부 | 재원 또는 비원 |
| **학교** | 선택 | 학교명 | 목동중 |
| **학부** | 선택 | 학부 구분 | 중등 또는 고등 |
| **학년** | 선택 | 학년 | 2 |
| **1주차** | 필수 | 1주차 시간표 | HA/월/16:00 |
| **2주차** | 필수 | 2주차 시간표 | GR/화/18:00 |
| **3주차** | 필수 | 3주차 시간표 | MA/수/17:00 |

### 주차별 시간표 작성 규칙

각 주차 칸에는 아래 형식으로 입력합니다:

```
레벨/요일/시간
```

**예시:**
- `HA/월/16:00` → HA 레벨, 월요일, 오후 4시
- `GR/화/18:00` → GR 레벨, 화요일, 오후 6시
- `MA/수/17:00` → MA 레벨, 수요일, 오후 5시

**주의사항:**
- 슬래시(/) 로 구분합니다
- 시간은 24시간 형식입니다 (오후 4시 = 16:00)
- 해당 주차에 수업이 없으면 칸을 비우거나 `-` 를 입력합니다

### CSV 예시 (엑셀에서 이렇게 보입니다)

| 이름 | 연락처 | 재원/비원 | 학교 | 학부 | 학년 | 1주차 | 2주차 | 3주차 |
|------|--------|----------|------|------|------|-------|-------|-------|
| 김민준 | 01012345678 | 재원 | 목동중 | 중등 | 2 | HA/월/16:00 | HA/수/16:00 | HA/금/16:00 |
| 이서연 | 01098765432 | 비원 | 진명여고 | 고등 | 1 | GR/화/18:00 | GR/목/18:00 | GR/토/14:00 |
| 박지호 | 01055556666 | 재원 | 목동중 | 중등 | 3 | MA/월/17:00 | - | MA/금/17:00 |

## 1단계: 로그인하기

1. 크롬 브라우저를 엽니다.
2. 주소창에 시스템 주소를 입력하고 Enter를 누릅니다.
3. 화면 **우측 상단**에 동그란 **G** 아이콘이 보입니다. 이것을 클릭합니다.
4. Google 로그인 화면이 나타납니다. **승인된 계정**의 이메일과 비밀번호를 입력합니다.
5. 로그인에 성공하면 **G** 아이콘이 내 이메일 첫 글자로 바뀝니다.
6. 학생 목록이 자동으로 화면에 나타나면 로그인 완료입니다.

---

## 2단계: 학기 확인하기

1. 화면 **왼쪽 사이드바** 맨 위에 **학기 선택** 드롭다운이 있습니다.
2. 드롭다운을 클릭하여 **현재 학기**를 선택합니다 (예: 2026-Spring).

---

## 3단계: 문법 특강 관리 화면 열기

1. 화면 **우측 상단 메뉴**에서 **비내신 문법 특강 관리** 버튼을 찾아 클릭합니다.
2. 화면 중앙에 **문법 특강 관리 모달**(팝업 창)이 열립니다.
3. 모달 안에 다음 항목들이 보입니다:
   - 시작일 / 종료일 / 주차 수 입력란
   - CSV 불러오기 버튼
   - 대시보드 영역 (학생 현황이 표시될 곳)
   - 닫기 / 저장 버튼

---

## 4단계: 특강 기간 설정하기

### 4-1. 시작일 입력
1. **시작일** 칸을 클릭합니다.
2. 달력이 나타납니다.
3. 문법 특강이 **시작되는 날짜**를 선택합니다.

### 4-2. 종료일 입력
1. **종료일** 칸을 클릭합니다.
2. 달력이 나타납니다.
3. 문법 특강이 **끝나는 날짜**를 선택합니다.

### 4-3. 주차 수 입력
1. **주차 수** 칸에 특강이 몇 주 동안 진행되는지 숫자를 입력합니다.
2. 기본값은 **3** 입니다.
3. 1~8 사이의 숫자를 입력할 수 있습니다.
4. CSV 파일의 주차 열 수와 맞춰야 합니다.
   - 예: CSV에 1주차, 2주차, 3주차 열이 있으면 → 주차 수 = 3

> **주의사항**
> - 종료일은 반드시 시작일보다 **이후 날짜**여야 합니다.
> - 기간을 설정하면 이미 등록된 문법 특강 학생이 있을 경우 자동으로 표시됩니다.

---

## 5단계: CSV 파일 불러오기

1. **CSV 불러오기** 버튼을 클릭합니다.
2. 파일 선택 창이 나타납니다.
3. 미리 준비한 **CSV 파일**을 찾아 선택합니다.
4. **열기** 버튼을 클릭합니다.

### 불러오기 성공 시
- 버튼 옆에 **"총 OO명 (재원 OO + 비원 OO)"** 이라는 요약이 표시됩니다.
- 아래 대시보드에 주차별 학생 현황이 나타납니다.

### 불러오기 실패 시
- **"시작일과 종료일을 먼저 설정해주세요"** → 4단계로 돌아가 기간을 먼저 설정하세요.
- **"CSV에 이름/연락처 컬럼을 찾을 수 없습니다"** → CSV 파일 헤더에 "이름"과 "연락처" 열이 있는지 확인하세요.
- **"CSV 데이터가 부족합니다"** → CSV 파일에 헤더와 데이터가 최소 2줄 이상 있어야 합니다.

> **팁**: CSV를 잘못 불러왔으면 다시 **CSV 불러오기** 버튼을 클릭하여 다른 파일을 선택하면 됩니다.

---

## 6단계: 대시보드에서 등록 현황 확인하기

CSV를 불러오면 대시보드에 **주차별 시간표**가 카드 형태로 표시됩니다.

### 대시보드 구성
```
┌─ 1주차 ───────────────────────────────────┐
│  level-HA / 월 / 16:00  [3명]             │
│  [김민준] [박지호] [이서연]                  │
│                                           │
│  level-GR / 화 / 18:00  [2명]             │
│  [최유진] [정하늘]                          │
├─ 2주차 ───────────────────────────────────┤
│  ...                                      │
├─ 3주차 ───────────────────────────────────┤
│  ...                                      │
└───────────────────────────────────────────┘
```

### 학생 칩(태그) 색상 구분
- **진한 색 칩** = 재원생 (이미 등록된 학생)
- **연한 색 칩** = 비원생 (새로 등록되는 학생)

### 확인할 사항
- [ ] 각 주차에 학생이 올바르게 배치되어 있는가?
- [ ] 레벨(HA, GR, MA 등)이 맞는가?
- [ ] 요일과 시간이 맞는가?
- [ ] 재원/비원 구분이 맞는가?
- [ ] 총 인원 수가 맞는가?

---

## 7단계: 저장하기

1. 대시보드에서 모든 내용을 확인한 후, 모달 하단의 **저장** 버튼을 클릭합니다.
2. **확인 대화상자**가 나타납니다:
   - "OO명 문법 특강을 저장합니다. (재원 OO + 비원 OO) 기간: YYYY-MM-DD ~ YYYY-MM-DD 진행하시겠습니까?"
3. **확인** 을 클릭하면 저장이 시작됩니다.

### 기존 특강과 겹칠 때
- 같은 기간에 이미 문법 특강이 등록된 학생이 있으면 경고 메시지가 나타납니다.
- **확인** → 기존 특강을 새 데이터로 덮어씁니다.
- **취소** → 저장을 중단하고 돌아갑니다.

### 저장 완료 후
- 버튼이 **"저장 중..."** 에서 다시 **"저장"** 으로 바뀌면 완료입니다.
- 모달을 닫아도 됩니다.

---

## 저장 후 알아두면 좋은 것들

### 저장된 데이터는 어떻게 되나요?
- **재원생**: 기존 학생 정보에 문법 특강 수업이 **추가**됩니다.
- **비원생**: 새 학생으로 **자동 생성**됩니다 (상태: "등원예정").
- 각 학생에게 아래 정보가 등록됩니다:
  - 수업종류: 특강
  - 레벨기호: GR (Grammar)
  - 반넘버: 901
  - 주차별 시간표 (weekly_schedule)

### 등록된 학생 확인하기
- 메인 화면에서 학생을 클릭하면 상세 패널에서 **특강** 수업이 추가된 것을 확인할 수 있습니다.
- 사이드바에서 **수업종류 → 특강** 필터를 클릭하면 특강이 등록된 학생만 볼 수 있습니다.

### 이미 저장한 특강을 수정하고 싶으면?
- 같은 기간으로 모달을 열면 이미 저장된 학생들이 자동으로 표시됩니다.
- CSV를 다시 불러와서 저장하면 **덮어쓰기**로 수정됩니다.

### 변경 기록
- 모든 변경사항은 **자동으로 기록**됩니다.

## 자주 하는 실수 & 해결법

| 실수 | 해결법 |
|------|--------|
| CSV 파일이 안 열려요 | 파일 형식이 .csv인지 확인. 엑셀에서 "다른 이름으로 저장" → CSV 형식 선택 |
| "이름/연락처 컬럼을 찾을 수 없습니다" | CSV 첫 줄에 "이름", "연락처" 라는 헤더가 정확히 있는지 확인 |
| 주차별 시간표가 안 나와요 | CSV에 "1주차", "2주차" 등의 헤더가 있는지, 데이터가 "레벨/요일/시간" 형식인지 확인 |
| 특정 학생이 특정 주차에 안 보여요 | 해당 학생의 해당 주차 칸이 비어있거나 "-"로 되어 있을 수 있음 |
| "모든 학생이 이미 저장되어 있습니다" | 같은 기간에 이미 저장한 데이터가 있음. CSV를 수정하고 다시 불러오세요 |
| 비원생이 재원으로 잘못 표시돼요 | CSV "재원/비원" 열의 값을 확인하세요. 이 열이 없으면 시스템이 자동 판별함 |
| 저장 후 학생을 찾을 수 없어요 | 사이드바에서 학기와 수업종류(특강) 필터를 확인하세요 |
| 주차 수를 잘못 입력했어요 | 주차 수를 수정하고 CSV를 다시 불러오면 됩니다 |

## 전체 흐름 요약

```
로그인 → 학기 선택 → 문법 특강 관리 열기
→ 기간 설정 (시작일/종료일/주차 수)
```
