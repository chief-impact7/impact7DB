# 내신 시간표 작성 매뉴얼

> 컴퓨터에 익숙하지 않은 선생님도 따라할 수 있도록 모든 단계를 세세하게 안내합니다.

---

## 시작하기 전에 확인할 것

1. **인터넷 연결** 확인 (와이파이 또는 유선)
2. **크롬 브라우저** 사용 권장 (인터넷 익스플로러는 안 됩니다)
3. **승인된 Google 계정**으로 로그인 가능한지 확인

---

## 1단계: 로그인하기

1. 크롬 브라우저를 엽니다.
2. 주소창에 시스템 주소를 입력하고 Enter를 누릅니다.
3. 화면 **우측 상단**에 동그란 **G** 아이콘이 보입니다. 이것을 클릭합니다.
4. Google 로그인 화면이 나타납니다. **승인된 계정**의 이메일과 비밀번호를 입력합니다.
5. 로그인에 성공하면 **G** 아이콘이 내 이메일 첫 글자(예: "김" → "K")로 바뀝니다.
6. 학생 목록이 자동으로 화면에 나타나면 로그인 완료입니다.

> **문제가 생겼나요?**
> - 로그인이 안 되면 → 승인된 계정인지 관리자에게 확인하세요.
> - 학생 목록이 안 보이면 → 왼쪽 사이드바에서 **학기**가 올바르게 선택되어 있는지 확인하세요.

---

## 2단계: 학기 확인하기

1. 화면 **왼쪽 사이드바** 맨 위에 **학기 선택** 드롭다운이 있습니다.
2. 드롭다운을 클릭하면 학기 목록이 나타납니다 (예: 2026-Spring).
3. **현재 학기**를 선택합니다.
4. 올바른 학기를 선택해야 해당 학기 학생들이 표시됩니다.

> **왜 중요한가요?**
> 학기가 잘못 선택되어 있으면 내신 시간표를 설정할 학생들이 보이지 않습니다.

---

## 3단계: 내신 시간표 설정 화면 열기

1. 화면 **우측 상단 메뉴**에서 **내신 시간표 설정** 버튼을 찾아 클릭합니다.
2. 화면 중앙에 **내신 시간표 설정 모달**(팝업 창)이 열립니다.

> 모달이란? 기존 화면 위에 뜨는 작은 창입니다. 이 창에서 모든 설정을 합니다.

---

## 4단계: 내신 기간 설정하기

모달 상단에 **시작일**과 **종료일** 입력 칸이 있습니다.

### 시작일 입력
1. **시작일** 칸을 클릭합니다.
2. 달력이 나타납니다.
3. 내신 수업이 **시작되는 날짜**를 선택합니다.
   - 예: 내신 대비가 4월 7일부터라면 → 4월 7일을 클릭

### 종료일 입력
1. **종료일** 칸을 클릭합니다.
2. 달력이 나타납니다.
3. 내신 수업이 **끝나는 날짜**를 선택합니다.
   - 예: 중간고사가 4월 25일에 끝난다면 → 4월 25일을 클릭

> **주의사항**
> - 종료일은 반드시 시작일보다 **이후 날짜**여야 합니다.
> - 날짜를 잘못 선택했으면, 다시 칸을 클릭하고 올바른 날짜를 선택하면 됩니다.

---

## 5단계: 학생 그룹 확인하기

기간을 설정하면 시스템이 **자동으로 학생들을 그룹**으로 나누어 보여줍니다.

### 그룹 구성 기준
- **학교** + **학부(중등/고등)** + **학년** 별로 묶입니다.
- 예: "진명여고 고등 2학년" 그룹, "목동중 중등 3학년" 그룹
- **초등학생은 제외**됩니다 (중등, 고등만 해당)
- **재원** 또는 **등원예정** 상태 학생만 포함됩니다

### 화면에서 확인하기
- 각 그룹 이름 아래에 해당 **학생 이름이 칩(작은 둥근 태그)** 형태로 나열됩니다.
- 학생이 빠져 있다면 → 해당 학생의 상태나 학부 정보가 올바른지 확인하세요.

---

## 6단계: 서브그룹별 요일과 시간 설정하기

각 학생 그룹 안에 **서브그룹**이 있습니다. 서브그룹은 같은 학교/학년이지만 **다른 요일/시간에 오는 학생들**을 나누기 위한 것입니다.

### 6-1. 요일 선택하기

1. 서브그룹 영역에 **월, 화, 수, 목, 금, 토** 체크박스가 있습니다.
2. 이 학생들이 내신 수업에 올 **요일을 체크**합니다.
   - 예: 월, 수, 금에 온다면 → 월, 수, 금에 체크
3. 체크하려면 해당 요일 옆의 **네모 칸을 클릭**합니다.
4. 다시 클릭하면 체크가 해제됩니다.

### 6-2. 시간 입력하기

1. **시간** 입력 칸을 클릭합니다.
2. 시간을 입력합니다.
   - 예: 오후 4시 → 16:00
   - 예: 오후 7시 30분 → 19:30
3. 시간 선택기가 나타나면 시와 분을 선택합니다.

> **팁**: 오전/오후 구분 없이 24시간 형식으로 입력합니다.
> - 오후 1시 = 13:00
> - 오후 2시 = 14:00
> - 오후 3시 = 15:00
> - 오후 4시 = 16:00
> - 오후 5시 = 17:00
> - 오후 6시 = 18:00
> - 오후 7시 = 19:00
> - 오후 8시 = 20:00

---

## 7단계: 서브그룹 추가/삭제 (필요한 경우)

같은 학교/학년 학생들 중에서 **다른 요일/시간에 오는 그룹**이 있다면 서브그룹을 추가합니다.

### 서브그룹 추가하기
1. 해당 그룹 영역 안에 있는 **서브그룹 추가** 버튼을 클릭합니다.
2. 새로운 서브그룹이 생기고, 요일과 시간을 별도로 설정할 수 있습니다.

### 서브그룹 삭제하기
1. 삭제하고 싶은 서브그룹의 **삭제** 버튼을 클릭합니다.
2. 해당 서브그룹이 제거됩니다.

> **예시 상황**
> "목동중 3학년" 학생이 10명인데:
> - 5명은 월/수 16:00에 오고
> - 5명은 화/목 18:00에 온다면
> → 서브그룹을 2개로 나눠서 각각 설정합니다.

---

## 8단계: 개별 학생 예외 처리 (필요한 경우)

그룹 전체와 다른 요일/시간에 오는 학생이 한두 명 있을 때 사용합니다.

### 예외 설정하기
1. 해당 학생의 **이름 칩(둥근 태그)을 클릭**합니다.
2. **예외 처리 팝업**이 나타납니다.
3. 이 학생만의 **요일**과 **시간**을 별도로 설정합니다.
4. 설정을 완료하면 팝업을 닫습니다.

> **언제 사용하나요?**
> - 그룹은 월/수 16:00인데, 김민준 학생만 화/목 17:00에 온다면
> - 김민준 칩을 클릭해서 화/목 17:00으로 따로 설정합니다.

---

## 9단계: 설정 내용 최종 확인하기

저장하기 전에 모든 설정을 다시 한번 확인합니다.

### 확인 체크리스트
- [ ] **시작일**과 **종료일**이 올바른가?
- [ ] 모든 **학생 그룹**이 빠짐없이 표시되어 있는가?
- [ ] 각 그룹의 **요일**이 올바르게 체크되어 있는가?
- [ ] 각 그룹의 **시간**이 올바르게 입력되어 있는가?
- [ ] 예외 처리할 학생이 있다면 **예외 설정**을 했는가?
- [ ] 서브그룹이 필요한 경우 **서브그룹 분리**를 했는가?

---

## 10단계: 저장하기

1. 모든 설정을 확인한 후, 모달 하단의 **저장** 버튼을 클릭합니다.
2. 시스템이 다음을 자동으로 확인합니다:
   - 기간이 올바른지 (종료일 > 시작일)
   - 모든 학생에게 요일이 설정되어 있는지
3. **기존 내신 시간표와 겹치는 기간**이 있으면 경고 메시지가 나타납니다.
   - "덮어쓰기" → 기존 시간표를 새 시간표로 대체
   - "취소" → 돌아가서 기간 수정
4. 저장이 완료되면 확인 메시지가 나타납니다.

> **저장 후 확인하기**
> - 메인 화면에서 학생을 클릭하면 상세 패널에서 **내신** 수업이 추가된 것을 확인할 수 있습니다.
> - 사이드바에서 **수업종류 → 내신** 필터를 클릭하면 내신 수업이 등록된 학생만 볼 수 있습니다.

---

## 저장 후 알아두면 좋은 것들

### 내신 기간 중 정규 수업은 어떻게 되나요?
- **자동으로 숨겨집니다.** 내신 기간 동안에는 정규 수업 대신 내신 수업이 표시됩니다.
- 내신 종료일이 지나면 **자동으로 정규 수업이 다시 나타납니다.**
- 선생님이 따로 조작할 필요가 없습니다.

### 잘못 설정했을 때
- 같은 기간으로 다시 내신 시간표를 설정하면 **덮어쓰기**로 수정할 수 있습니다.
- 개별 학생의 수업은 학생 상세 패널에서 직접 수정할 수도 있습니다.

### 변경 기록
- 모든 변경사항은 **자동으로 기록**됩니다.
- 누가 언제 어떤 변경을 했는지 추적할 수 있습니다.

---

## 자주 하는 실수 & 해결법

| 실수 | 해결법 |
|------|--------|
| 학생이 목록에 안 보여요 | 학기가 올바르게 선택되어 있는지 확인. 학생 상태가 "재원" 또는 "등원예정"인지 확인 |
| 종료일을 시작일보다 앞에 넣었어요 | 시스템이 자동으로 막아줍니다. 날짜를 다시 설정하세요 |
| 요일을 안 넣고 저장했어요 | 시스템이 저장을 막고 경고를 표시합니다. 요일을 체크한 후 다시 저장하세요 |
| 특정 학생만 다른 시간에 와야 해요 | 학생 이름 칩을 클릭해서 예외 처리하세요 |
| 내신 기간이 끝났는데 정규가 안 보여요 | 종료일이 아직 지나지 않았는지 확인하세요 |
| 이미 만든 내신 시간표를 수정하고 싶어요 | 같은 기간으로 다시 설정하면 덮어쓰기 가능합니다 |

---

## 전체 흐름 요약

```
로그인 → 학기 선택 → 내신 시간표 설정 열기 → 기간(시작일/종료일) 입력
→ 자동 그룹 생성 확인 → 서브그룹별 요일/시간 설정
→ (필요시) 서브그룹 추가 → (필요시) 개별 학생 예외 처리
→ 최종 확인 → 저장
```
